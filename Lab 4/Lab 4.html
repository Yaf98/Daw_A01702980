Preguntas sobre JavaScript

¿Qué diferencias y semejanzas hay entre Java y JavaScript?[1]
  • JavaScript es un lenguaje desarrollado por Netscape, Java fue desarrollado por James Goslin
  • JavaScript no forma parte de Java, ni tampoco es una extension de él
  • Usualmente JavaScript es usado para darle interactividad a los sitios web
  • Las aplicaciones creadas en JavaScript corren en navegadores web, mientras que las aplicaciones de Java corren en una maquina virtual
  • El código JavaScript necesita ser interpretado. Por otro lado, el código de Java se compila
¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

El objeto Date muestra la hora de acuerdo con la configuración de la zona del navegador. Algunos de sus métodos son[2] :
  1. getTime()
  2. getDay()
  3. getHours()
  4. getMonth()
  5. getSeconds
  6. getDate()
  7. getFullYear()
¿Qué métodos tienen los arreglos? (Menciona al menos 5*)[3]
  1. filter()
  2. findIndex()
  3. from()
  4. concat()
  5. forEach()
  6. entries()
¿Cómo se declara una variable con alcance local dentro de una función?[4]
  • Para declarar una función con alance local sólo basta anteponer la palabra reservada "let"
  • Por ejemplo: let nombre_variable
¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
  • Las variables globales en las funciones pueden ser sobre escritas por otros scripts[5]
¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?
  • El método match() se utliza para buscar strings que coincidan con expresiones regulares. Las coincidencias encontradas con retornas en un arreglo de objetos y en caso de que no se encuentre ninguna, se retorna null [6]
  • Pienso que se podrían utiliazr expresiones regulares para buscar cadenas de texto en una página, de la misma manera que ocurre cuando presionamos "Ctrl F" en el navegador para buscar palabras. Considero que también se podría utilizar para al auto completado de palabras
