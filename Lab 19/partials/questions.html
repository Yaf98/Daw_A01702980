Preguntas

¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications? [1]
  • Las aplicaciones RIA buscan conbinar las ventajas de las aplicaciones de escritorio junto con las aplicaciones web. Usar AJAX es apropiado cuando queremos que la aplicación se comunica con el cliente de manera asíncrona , sin la necesidad de recargar toda la página para mejor experiencia de usuario
¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor? [2]
  • Las validaciones deben hacerse del lado del cliente. No deben hacerse del lado del servidor a menos que se requiera de conectarse a una base de datos para extraer información.
