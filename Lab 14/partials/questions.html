Preguntas

¿Qué es ODBC y para qué es útil? [1]
  • Open Data Base Connectivity (ODBC) es una API de estándar abierto para acceder a una base de datos
  • Su objetivo es hacer posible que desde cualquier aplicación sea posible acceder a los datos sin importar el gestor en el que esten
¿Qué es SQL Injection? [2]
  • Es un tipo de ataque hacía una base de datos
  • Es una técnica de hacking web bastante utilizada
  • SQL Injection consiste en insertar software malicioso por medio SQL statments, esto a tráves del input en una página web
¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection? [2]
  • Para prevenirnos de estos ataques podemos usar prepared statments SQL
  • Estos checan cada parámetro dado para verficar que esten en la forma correcta para cada columna y no sean parte de una sentencia SQL ejecutable
