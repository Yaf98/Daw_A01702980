Preguntas

¿Por qué es una buena práctica separar el controlador de la vista? [1]
  • Desarrollar en paralelo. Debido a que la vista esta separada de la lógica es posible que dos personas puedan trabajar simultanéamente en diferentes partes
  • Modificaciones en alguna parte del proyecto no afecta las otras partes
Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función? [2]
  • $_GLOBALS
    • Array que contiene las referencias a todas las variables globales del script. Las claves del array son el nombre de las variables.
  • $_SERVER
    • Array que contiene cabeceras, rutas y ubicaciones del script
  • $_FILES
    • Es un array asociativo de elementos subidos al script que sé está ejecutando, a tráves del método POST. Contiene atributos de los archivos como pueden ser : nombre, tamaño, tipo, etc.
  • $_COOKIE
    • Array asociiativo de variables pasadas al script actual a través de Cookies HTTP
  • $_SESSION
    • Array asociativo que ccontiene variables de sesión disponibles para el script actual
  • $_REQUEST
    • Array asociativo que contiene el contenido de $_GET, $_POST y $_COOKIE
  • $_ENV
    • Similar a _$COOKIE, contiene variables pasadas al script a tráves del entorno

¿Qué es XSS y cómo se puede prevenir?[5]
  • Cross-site scripting (XSS) se refiere a un tipo de ataque donde el atacante introduce código malicioso del lado del cliente
  • Se puede prevenir liberando los outputs con htmlspecialchars() o htmlentities(). Estas funciones caracteres problemáticos en entidades HTML provocando que el código insertado sea un output inofensivo
