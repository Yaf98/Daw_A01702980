Cuestionario 1: BD vs DBMS

¿Qué es entonces una base de datos?
  • Es una colección de archivos interrelacionados que contiene información relevante para una organización [1]

¿En qué casos es conveniente usar bases de datos?[2]
  • Cuando nuestra información está contenida en archivos interrelacionados y usuarios o aplicaciones necesitan acceder a ella eficientemente
  • También cuando tenemos un número grande de registros, importa si el conjutno de datos es pequeño

¿Qué es un sistema de gestión de base de datos - DBMS?
  • Un sistema de gestión de bases de datos consiste en una colección de archivos relacionados entre sí, pero no sólo eso, sino que también consiste en un conjunto de programas que nos permiten manipular y acceder a los datos eficientemente. [1]

Enlista y explica las funciones/responsabilidades que tienen los DBMS. [2]
  • Implantación de la integridad: tener claras las restricciones que se tienen que satisfacer y analizar si en algún momento se violan.
  • Implantación de la seguridad: hacer cumplir requisitos de seguridad, es decir, quien tiene acceso a qué información, dependiendo del rol que desempeñen para la utilización de los datos.
  • Copia de seguridad y recuperación: detectar cualquier fallo y restaurar la base de datos de manera que quede en el estado que existía antes de ocurrir el fallo.
  • Control de concurrencia: conservar la consistencia de los datos mediante el control de la interacción entre usuarios concurrentes.
  • Definir estructuras para el almacenamiento de información
  • Definir procedimientos para manipular los datos
  • Interactuar con el gestor de archivos. Traduce sentencias DML a comandos del sistema de archivos de bajo nivel

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
  • En el proyecto de Diseño Interactivo pensábamos hacer una página web en la cual se pudiera guardar cualquier tipo de información valioso para el usuario. Podría guardar en un muro personal recursos que le parecieran útiles como vídeos, enlaces a páginas, web, libros, etc. Hubiera sido conveniente utilizar una base de datos para guardar la información relativa a cada usuario, permitiendo así recuperar su información y mantener actualizados los cambios en todo momento.
